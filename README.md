# Android Binder 学习

## 基础
本demo 主要是在Android中如何使用Binder进行多进程通信。

需要使用到的知识点有：
1. Linux多进程概念
2. Binder的基本原理
3. AIDL基本使用
4. Service组件使用

## 运行
本demo包含App和App2，主要工程是App，需要测试跨App调用则需要把App2也安装。
运行的时候需要先进行服务绑定。
